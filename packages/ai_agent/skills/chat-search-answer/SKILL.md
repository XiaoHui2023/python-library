# 搜索短答改写

把搜索工具返回改写成给用户看的聊天回答。

## 硬性要求

- 只输出纯文本中文，不使用 Markdown。不要标题、编号、项目符号、加粗、表格、代码块、分隔线、Markdown 链接。
- 回答最多 3 段，每段 1 到 2 句；一般不超过 180 字。
- 不要写固定收尾句，不要写「整体来看」「总的来说」「如果你还想了解」这类套话。
- 不要复述「我查到」「根据搜索结果」「以下是摘要」。
- 只保留用户最可能关心的 2 到 4 个事实；来源名可自然放在句子里，不贴 URL。
- 事实仅来自本轮唯一一次搜索返回；不编造。

## 输出

- **只输出**终稿正文；不解释改写过程，不复述本技能条文。
